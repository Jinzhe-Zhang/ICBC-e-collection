### jar包说明
commons-logging-1.2.jar----------------与日志记录相关
icbc-api-sdk-cop.jar-------------------sdk基础jar包
icbc-api-sdk-cop-io.jar----------------sdk各接口输入输出
icbc-ca.jar----------------------------与工行CA认证相关
InfosecCrypto_Java1_02_JDK14+.jar------与工行CA认证相关

### jar包使用说明
1.支持java版本1.5及以上
2.icbc-api-sdk-cop.jar、icbc-api-sdk-cop-io.jar为必须jar包
3.如果认证方式不是工行的CA证书认证，可以不引入icbc-ca.jar和InfosecCrypto_Java1_02_JDK14+.jar
